腾讯-罗盘BI可视化平台

2013年4月1日
| 阅读

角色

负责人之一，负责多个模块的设计与开发

背景

前身是OZ(Operation Zone),系统太旧,不能满足数据可视化、邮件推送、数据体系建设、数据模型规范化的需求
为支持开放平台app数据分析服务，开发了腾讯外网罗盘和内网罗盘
外网罗盘更模板化、统一化，服务第三方开发商；内网罗盘更个性化、自助化，服务内部各个产品

开发工具

  • 后台开发语言php5,cakephp,phpmailer
  • 前端开发语言html5,css3,js
  • js框架bootstrap,jquery,highcharts,phantomjs,zTree,DataTables,echarts,multiselect
  • 数据存储postgresql,mysql5
  • 配置缓存memcache
  • 数据缓存redis
  • 负责均衡nginx + keepalived + vip
  • 四层负载LVS(后废弃)
  • 共享存储tfs

功能模块

  • 数据源管理
  • 产品管理
  • 目录管理
  • 报表自助配置
  • 报表可视化
  • 大屏管理
  • 个人收藏管理
  • 邮件订阅、指标推送
  • 报表、邮件、指标搜索
  • 报表水印，邮件水印
  • 报表笔记管理
  • 指标异动告警
  • 数据模型配置管理
  • 自定义组合报表
  • 自定义数据对比
  • OLAP多维分析
  • 指标池管理
  • 消息管理
  • 下载管理
  • 维表管理
  • 微信公众号
  • 短网址服务
  • API接口服务
  • 权限管理
  • ABTest
  • 活动分析
  • wiki帮助
