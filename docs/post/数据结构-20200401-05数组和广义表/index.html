二、矩阵的压缩存储

在高阶矩阵中，有许多值相同的元素或者是零元素，有时为了节省存储空间，可以对这类矩阵进行压缩存储。  
压缩存储：指的是为多个值相同的元只分配一个存储空间，对零元不分配空间  
特殊矩阵：假若值相同的元素或者零元素在矩阵中的分布有一定的规律  
稀疏矩阵：没有规律
