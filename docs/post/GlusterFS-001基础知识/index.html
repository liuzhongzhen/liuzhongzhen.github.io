二、安装指南

  1. rpm -qa|grep gluster 查看机器上面已经安装的gluster版本
  2. yum remove gluster* 移除已经安装的gluster老的版本
  3. yum search gluster 搜索gluster yum源
  4. yum install centos-release-gluster7.noarch 安装gluster7 yum源
  5. yum install glusterfs glusterfs-api glusterfs-cli glusterfs-fuse glusterfs-libs glusterfs-server glusterfs-geo-replication glusterfs-rdma 服务器安装组件
  6. yum install glusterfs glusterfs-fuse attr -y 客户端安装组件

  8. service iptables save 保存iptable配置
  9. systemctl stop rpcbind.service 停止rpcbind服务
  10. systemctl restart iptables.service 重启iptable服务
  11. systemctl enable glusterd.service #开机自动启动文件系统服务
  12. systemctl start glusterd.service #启动服务
  13. systemctl list-units –type=service 列出开机启动的服务

基本操作:
a. 查看卷状态 gluster volume status
b. 不停机的情况下扩展卷
gluster volume add-brick glustervol1  wxtest064vm15:/data/gluster_brick2 wxtest064vm16:/data/gluster_brick2 wxtest064vm17:/data/gluster_brick2  
c. 查看卷信息 gluster volume info gluster_vol1
d. 重新平衡扇区 gluster volume rebalance gluster_vol1 start
e. 查看重新平衡的状况 gluster volume rebalance gluster_vol1 status
f. 强制重启gluster volume start gluster_vol1 force
数据是hash分布的，并且是复制的
系统管理分成：节点管理，卷管理，brick管理
设置 gluster volume set gluster_vol1 performance.client-io-threads on
设置nfs打开 gluster volume set gluster_vol1 nfs.disable off

客户端对 GlusterFS 设置ACL规则
设置其他用户对目录有读写权限：
setfacl -m o::rwx -R arfa_nfs/
setfacl -m d:o:rwx arfa_nfs/ 

mount -t glusterfs -o acl 10.33.35.116:/gluster_vol1 /mnt/gluster_test

杀死进程：
gluster volume stop gluster_vol1
systemctl stop glusterd.service 
systemctl disable glusterd.service
